# Photobooth Web Application

## Project Overview

This project is a web-based photobooth application that allows users to select creative frames and capture photos directly from their browser.

## Current Progress

At this stage, the following has been completed:

- Project structure setup
- Creation of all main HTML pages:
  - index.html (Landing page)
  - camera.html
  - about.html
  - contact.html
  - privacy.html
- Basic layout structure for each page
- Initial image assets added (frames, logo, etc.)

## Pending Work

The following features are still under development:

- CSS styling and visual design
- JavaScript functionality (camera activation, photo capture, template logic)
- Responsive design implementation
- Animations and UI enhancements

## Technologies Used

- Git & GitHub
- GitHub Pages (deployment planned)
